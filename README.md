### 구현된 기능
- **로그인**<br>
- **QR읽어 입고 처리**<br>
- **같은 QR 중복 처리**<br>
- **입고되지 않은 QR은 출고 미처리**<br>
- **입고된 QR만 출고 가능**<br>

### SSL 적용
- **웹에서 카메라 기능을 사용 할 경우 https 프로토콜에서만 카메라 동작 가능**
